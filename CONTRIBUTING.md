\# Guia de Contribució



\## Estil de codi

\- Utilitza sangria de 2 espais.

\- Escriu comentaris clars per a funcions i classes.

\- Segueix la nomenclatura CamelCase per a classes i snake\_case per a variables.



\## Pull Requests

\- Fes pull requests cap a la branca `main`.

\- Intenta que cada PR tingui un objectiu clar i commits petits.

\- Revisa que tots els tests passin abans de fer merge.



\## Report d'Issues

\- Cerca si l'issue ja existeix abans de crear-ne un de nou.

\- Sigues clar i detallat: descriu els passos per reproduir el problema, el que esperes i el que passa.

\- Assigna labels si s’escau.
